Hand-drawn concept hero (Track B)

The three things that make a hand-drawn hero look designed: ① a hand-lettered font (Shantell Sans — handwriting only renders for Latin labels) ② the hub has an inner routed loop, not an empty box ③ the strokes get a soft glow over a dark vignette. Labels are English so the handwriting reads — which is also the limit: this full look only works for English / conceptual diagrams; CJK labels fall back to a clean font (use Track A for those).
